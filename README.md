**REGARDING PHISHING EMAIL DETECTION AND AWARENESS SYSTEM**
The objective of this project is to analyze a suspicious email and identify phishing indicators that could compromise a user's personal or financial information.
The project aims to improve awareness of common phishing techniques and demonstrate methods for detecting fraudulent emails.

**TOOLS Used**

Gmail email template 
microsoft word
Email textbook detector
web browser 

**ANALYSIS APPROACH**
1. Sender email verification
2. Subject line analysis
3. concept inspection
4. URL analysis
5. Risk assesment
6. Final conclusion

**OUTCOMES**:
Identify spoofed email addresses.
Recognize phishing URLs and fake domains.
Understand social engineering techniques used in phishing.
Analyze email headers and message content for security threats.
Apply cybersecurity best practices to avoid phishing attacks.
